package.json - Quản lý dependencies và scripts cho dự án React + TypeScript + Vite
vite.config.ts - Cấu hình Vite với React SWC plugin và Tailwind CSS
tsconfig.json - Cấu hình TypeScript chính (tham chiếu đến app và node configs)
tsconfig.app.json - Cấu hình TypeScript cho source code ứng dụng
tsconfig.node.json - Cấu hình TypeScript cho build tools (Vite)
tailwind.config.js - Cấu hình Tailwind CSS cho styling
eslint.config.js - Cấu hình ESLint cho code quality
index.html - Entry point HTML của ứng dụng
.gitignore - Loại trừ files/folders khỏi Git
🚀 Core Application
src/main.tsx - Entry point React, render App component
src/App.tsx - Component gốc, render AppRouter
src/index.css - Import Tailwind CSS directives
src/vite-env.d.ts - Type definitions cho Vite
🧭 Routing System
src/router/AppRouter.tsx - Cấu hình React Router với Layout, Error boundary
src/constants/routes.ts - Định nghĩa constants cho routes và labels
src/hooks/useAppNavigation.ts - Custom hook cho navigation với type safety
🏗️ Layout & Structure
src/Layout.tsx - Layout component chính (Header, Breadcrumb, Main, Footer)
📄 Pages
src/assets/pages/Home.tsx - Trang chủ với banner và danh sách game categories
src/assets/pages/Login.tsx - Trang đăng nhập với form validation
🧩 Header Components
src/assets/header/Header.tsx - Component header chính
src/assets/header/MainHeader.tsx - Header bar với logo, search, login/register buttons
src/assets/header/Topbar.tsx - Top bar với links (Sell, Help, Currency, Language)
src/assets/header/CategoryNav.tsx - Navigation bar cho categories (Currency, Items, Accounts, etc.)
🎨 UI Components
src/assets/banner/MainBaner.tsx - Banner chính với background và action buttons
src/assets/footer/Footer.tsx - Footer với payment methods, links, newsletter signup
🧭 Navigation Components
src/components/Navigation/Navigation.tsx - Reusable navigation component với icons
src/components/Breadcrumb/Breadcrumb.tsx - Breadcrumb navigation với route tracking
src/components/NavigationTest/NavigationTest.tsx - Test component cho navigation functionality
